Za pokretanje potrebno je napraviti virtualno okruženje sa instaliranim bibliotekama iz requirements.txt.
Kako bi se pokrenuo glavni koristi se naredba **python main.py**.
Kako bi se ispisala statistika za rezultate simulacije koristi se naredba **python plot_results.py**.
Kako bi se pokrenula datoteka koja radi više simulacija sa raznim parametrima koristi se linija **python experiments.py**.
Kako bi se ispisala statistika svih tih simulacija koristi se linija **python plot_experiments.py**.
